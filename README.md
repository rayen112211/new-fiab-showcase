# NEW FIAB - Precision Metal Components

A premium showcase website for NEW FIAB, an Italian manufacturer of precision metal components for luxury fashion brands.

## About

NEW FIAB specializes in high-precision CNC machining and metal component manufacturing for the world's most prestigious fashion houses. This website showcases our capabilities, services, and commitment to excellence.

## Technologies

- **React 18** - Modern UI library
- **TypeScript** - Type-safe development
- **Vite** - Fast build tool and dev server
- **Tailwind CSS** - Utility-first styling
- **Framer Motion** - Smooth animations
- **shadcn/ui** - Premium UI components

## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
